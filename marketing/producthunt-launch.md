## First Comment (Maker Story)

Hey PH! I'm a plumber by trade. No CS degree, no funding, no team.

Last year I got tired of paying $150+/mo for a handful of AI chatbots that couldn't do anything useful autonomously. So I started building.

The core insight was embarrassingly simple: 90% of what we ask AI to do is text processing. Bash handles that for free, instantly. So I built a pre-processing layer that routes simple tasks through bash pipes before any token ever hits a model.

The result: 15 specialized AI agents running on my laptop for $0/month.

- Captain handles customer FAQ
- Mitch closes sales leads
- Ghost runs social media on autopilot
- JobSite generates invoices
- 11 more, each with a specialty

Today I'm shipping Botwave Desktop — a native app to command the whole fleet from one window. Live status, one-click deploys, a script factory with 80+ reusable automations, and a model router so you can assign any agent to local or free cloud models.

Everything is open source. The models are free (LM Studio local + Groq/OpenRouter cloud). Your data never leaves your machine.

I built this on an old laptop with bash scripts. If a plumber can do it, anyone can.

Would love your feedback. Try the free demo: message @CaptainObvious_bot on Telegram.

## Key Features

1. **$0 API Cost** — Bash-pipe preprocessing handles 90% of tasks before models are called. Local models + free cloud fallback for the rest.

2. **15+ Specialized Agents** — Sales, content, research, invoicing, social media, analytics. Each agent has a focused role and personality.

3. **Desktop Fleet Commander** — Native app with live agent health, one-click deploy from 15+ templates, script factory, and visual model router.

4. **Phone Control** — Manage your entire fleet from Telegram. Deploy agents, check status, approve actions — all from your phone.

5. **Full Audit Trail** — Every command, conversation, and action logged. Complete transparency on what your agents do.
