Flash & Setup

Flash the SD card

Download the Botwave Node image and flash it using Raspberry Pi Imager or dd:

wget https://botwave.app/downloads/botwave-node-v1.img.gz
gunzip botwave-node-v1.img.gz
sudo dd if=botwave-node-v1.img of=/dev/sdX bs=4M status=progress

Boot & connect

Insert SD card, connect ethernet, power on. The Node broadcasts as botwave-node.local on your network.

ssh botwave@botwave-node.local
# Default password: changeme (you'll be prompted to change on first login)

Run the setup wizard

The one-click installer configures everything:

curl -sSL https://botwave.app/install.sh | bash

Connect Telegram

The wizard prompts for your Telegram bot token. Paste it in and your fleet goes live.

Join the mesh (optional)

Connect your Node to your Tailscale network for remote access from anywhere:

sudo tailscale up --authkey=YOUR_KEY

3D-Printed Case

Print your own Botwave Node case. STL files optimized for PLA at 0.2mm layer height.

botwave-case-top.stl

Top shell with ventilation slots and Botwave logo emboss. Print time: ~2hr.

botwave-case-bottom.stl

Bottom tray with Pi 5 mounting posts, SD card access slot, and rubber feet recesses. Print time: ~1.5hr.

botwave-fan-mount.stl

30mm fan bracket that clips into top shell. Keeps CPU under 60C under sustained load.

Print Settings

PLA, 0.2mm layers, 20% infill, no supports needed. Recommended color: black with amber accents.

FAQ

Can a Raspberry Pi actually run AI models?

Yes. The Pi 5 with 8GB RAM runs quantized 7B models (GGUF Q4) at ~5 tokens/sec via llama.cpp. For heavier tasks, the Model Router automatically falls back to free cloud APIs (Groq, Cerebras).

What's included in the pre-flashed image?

Kali Linux ARM, OpenClaw, LM Studio server, Botwave Desktop, Tailscale, 5 starter agents (Captain, Scout, Scribe, Ghost, ContentBot), and the full Script Factory library.

Can I access it remotely?

Yes — Tailscale mesh VPN is pre-installed. Access your Node from anywhere. Control agents via Telegram from your phone.

What about WiFi penetration testing?

The Pi 5 supports external USB WiFi adapters for monitor mode. Dick Tracy agent has full wireless audit capabilities (hcxdumptool + hashcat).

Do I need to update it?

Updates are automatic via the Botwave Desktop auto-updater. System updates run weekly via unattended-upgrades.

Can I sell Nodes to my clients?

Yes — the Operator Tier ($99/mo) includes white-label rights. Flash Nodes with your branding and sell as managed AI infrastructure.
